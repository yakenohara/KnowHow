# 文字列の表現

## シングルクォートで括った文字列



| `'` で括った文字列内                | `"` で括った文字列内                 |
| ----------------------------------- | ------------------------------------ |
| `"` はエスケープ無しで記述できる    | `"` はエスケープ無しで記述できる     |
| `'` を表現するには、`''` と記述する | `"` を表現するには、`'""` と記述する |
| 変数は展開されない                  | 変数は展開される                     |
|                                     |                                      |

## エスケープシーケンス
| 文字       | 表現      |
| ---------- | --------- |
| 改行(LF)   | `` `n``   |
| 改行(CR)   | `` `r``   |
| 改行(CRLF) | `` `r`n`` |
