JavaScriptのソースコードから 制御フロー図を DOT 形式で出力する Node.js 用ライブラリ `esgraph` と、  
ブラウザ上で graphviz を実行するための JavaScript ライブラリ `viz.js` を、  
同時にブラウザ上で実行する実装例。  

viz.js は 1.x 系と 2.x 系で実装方法が異なるので注意。
