# 設定ファイル

筆者環境だと、以下。  
この内、 `settings.txt` 内にアプリ設定がテキストで記載されている。  

```
C:\Users\<username>>\AppData\Roaming\CUE Tools
┣LocalDB.xml.z
┣motd.txt
┣profile-convert.txt
┣profile-fix.txt
┣profile-verify.txt
┗settings.txt
```

## おすすめ設定

v.2.2.6 におけるおすすめ設定を `settings/favorites` 配下に保存。  
デフォルトとの違いは、`settings/default` との違いを参照。
